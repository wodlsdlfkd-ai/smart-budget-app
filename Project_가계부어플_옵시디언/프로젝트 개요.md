# 📊 스마트 가계부 프로젝트 옵시디언

## 프로젝트 개요
카드 결제 알림을 자동으로 수집하여 구글 스프레드시트에 기록하고, 카드별 실적 현황을 한눈에 보여주는 자동화 가계부 시스템

## 아키텍처
```
📱 카드 결제 → 🔔 결제 알림 → 📲 안드로이드 앱 (알림 감지)
    → ⚙️ Google Apps Script (웹훅 수신 + Gemini AI 분류)
    → 📊 구글 스프레드시트 (자동 기입)
    → 🖥️ 웹 대시보드 PWA (시각화)
```
