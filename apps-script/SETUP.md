# 📖 Google Apps Script 셋업 가이드

이 문서는 스마트 가계부의 백엔드인 Google Apps Script를 기존 스프레드시트에 연결하는 방법을 안내합니다.

---

## 📋 사전 준비

1. 기존에 사용하던 **가계부 구글 스프레드시트** (빈양식 탭이 있는 것)
2. **Gemini API 키** (카테고리 자동 분류에 필요)

---

## Step 1: Gemini API 키 발급

1. [Google AI Studio](https://aistudio.google.com/apikey)에 접속합니다
2. 구글 계정으로 로그인합니다
3. **"API 키 만들기"** 버튼을 클릭합니다
4. 생성된 API 키를 복사해 둡니다 (나중에 코드에 입력)

---

## Step 2: Apps Script 에디터 열기

1. 가계부 **구글 스프레드시트**를 엽니다
2. 상단 메뉴에서 **확장 프로그램 → Apps Script** 를 클릭합니다
3. 새 탭에 Apps Script 에디터가 열립니다

---

## Step 3: 코드 붙여넣기

1. 에디터에 기본으로 있는 `function myFunction() {}` 코드를 **모두 삭제**합니다
2. [Code.gs](file:///c:/Users/wodls/Desktop/안티그래비티/가계부 어플/apps-script/Code.gs) 파일의 내용을 **전체 복사**하여 붙여넣습니다
3. 코드 상단의 설정 부분을 수정합니다:

```javascript
// 이 부분을 Step 1에서 발급받은 API 키로 교체하세요
const GEMINI_API_KEY = '여기에_API_키_붙여넣기';
```

4. **Ctrl + S** 또는 💾 아이콘을 클릭하여 저장합니다

---

## Step 4: 웹 앱으로 배포

1. 에디터 우측 상단의 **"배포" → "새 배포"** 를 클릭합니다
2. 유형 선택에서 **⚙️ 톱니바퀴 → "웹 앱"** 을 선택합니다
3. 설정:
   - **설명**: `스마트 가계부 API`
   - **다음 사용자 인증정보로 실행**: `나` (본인 계정)
   - **액세스 권한이 있는 사용자**: `모든 사용자`
4. **"배포"** 버튼을 클릭합니다
5. 처음 배포 시 **권한 승인**이 필요합니다:
   - "액세스 승인" 클릭
   - 본인 구글 계정 선택
   - "고급" → "안전하지 않은 페이지로 이동" 클릭 (본인이 만든 스크립트이므로 안전)
   - "허용" 클릭
6. **배포된 URL**이 표시됩니다 → 이 URL을 복사해 둡니다!

> [!IMPORTANT]
> 이 URL이 웹훅 주소입니다. 안드로이드 앱과 웹 대시보드 설정에 이 URL을 입력하게 됩니다.
> 
> 형식 예시: `https://script.google.com/macros/s/AKfycb.../exec`

---

## Step 5: 테스트

### 방법 A: Apps Script 에디터에서 직접 테스트

1. 에디터 상단 함수 선택 드롭다운에서 `testAddTransaction`을 선택합니다
2. **▶ 실행** 버튼을 클릭합니다
3. 실행 로그(하단)에서 결과를 확인합니다
4. 스프레드시트를 열어 **2026.06** 시트가 자동으로 생성되고 데이터가 기입되었는지 확인합니다

### 방법 B: 브라우저에서 API 테스트

배포된 URL 뒤에 파라미터를 붙여 브라우저에서 직접 호출:

```
https://script.google.com/macros/s/YOUR_ID/exec?action=summary&year=2026&month=6
```

---

## Step 6: 코드 수정 후 재배포

코드를 수정한 후에는 **반드시 재배포**해야 변경 사항이 반영됩니다:

1. **"배포" → "배포 관리"** 클릭
2. 기존 배포 옆의 **✏️ 연필 아이콘** 클릭
3. **버전**: "새 버전" 선택
4. **"배포"** 클릭

---

## 🔧 스프레드시트 컬럼 매핑

코드는 기존 스프레드시트의 아래 컬럼 구조에 맞춰져 있습니다:

| 열 | 컬럼 | 내용 | 자동 기입 |
|---|---|---|---|
| A | 1 | 날짜 | ✅ |
| B | 2 | 지출금액 | ✅ |
| C | 3 | 분류 | ✅ (Gemini AI) |
| D | 4 | 내역 | ✅ |
| E | 5 | 통장잔액 | ❌ (기존 수식 유지) |
| F | 6 | 카드사 | ✅ |

> [!TIP]
> 만약 기존 시트의 컬럼 순서가 다르다면, `Code.gs` 상단의 `COLUMNS` 객체에서 숫자만 변경하면 됩니다.

## ❓ 자주 묻는 질문

**Q: 비용이 발생하나요?**
A: Google Apps Script는 무료입니다. Gemini API도 무료 티어가 있어 일상적 사용량에서는 비용이 발생하지 않습니다.

**Q: 스프레드시트를 공유하면 다른 사람도 접근할 수 있나요?**
A: 배포된 URL을 아는 사람만 접근할 수 있습니다. URL을 공유하지 않으면 안전합니다.

**Q: 기존 데이터에 영향이 있나요?**
A: 없습니다. 기존 시트 탭의 데이터는 건드리지 않으며, 새로운 결제 내역만 빈 행에 추가됩니다.
